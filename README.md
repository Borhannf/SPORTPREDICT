 ⚽ SPORTPREDICT

SPORTPREDICT is a responsive sports prediction website where users can explore sports categories, view upcoming matches, and submit match score predictions.

 🚀 Features

- Responsive and modern UI
- Sports categories section
  - Football
  - Basketball
  - Tennis
  - Volleyball
- Match prediction system
- Interactive match cards
- Sign In / Sign Up page
- Mobile-friendly layout
- Ready for backend and API integration

 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript

 📂 Project Structure

```text
SPORTPREDICT/
│
├── index.html
├── sing-in.html
├── style .css
├── style-singin.css
└── README.md
```

 🎮 How It Works

1. Open the homepage.
2. Browse sports categories.
3. Select a match.
4. Enter your predicted score.
5. Submit your prediction.

